# Garment Trading App - Module & Feature Specification

## 1. User Management Module

### Features:
- Mobile number based login (OTP authentication)
- Role-based access:
  - Customer
  - Vendor
  - Admin
- User profile management:
  - Name
  - Phone number
  - Business name
- Account status:
  - Active / Inactive

---

## 2. Vendor Catalog Module

### Features:
- Add new product:
  - Multiple images upload
  - Category (e.g., Shirts, Jeans)
  - Attributes (pattern, fabric, color, etc.)
  - Price (optional)
  - MOQ (Minimum Order Quantity)
- Edit product details
- Delete product
- Mark product as Active / Inactive
- Bulk upload (future scope)

---

## 3. Customer Search & Discovery Module

### Features:
- Keyword-based search
- Filters:
  - Category
  - Pattern (Checkered, Plain)
  - Fabric
  - Price range
- Product listing:
  - Image-first layout
  - Quick preview
- Product detail page:
  - Images
  - Attributes
  - MOQ
  - Price

---

## 4. Cart & Order Request Module

### Features:
- Add products to cart
- Enter required quantity (no stock validation)
- Edit/remove items in cart
- Place order request
- View order summary before submission

---

## 5. Order Management Module

### Features:
- Auto split orders vendor-wise
- Maintain order lifecycle:
  - Pending
  - Accepted
  - Partially Accepted
  - Rejected
- Store order details:
  - Product
  - Vendor
  - Quantity requested
  - Quantity approved

---

## 6. Vendor Response Module

### Features:
- View incoming order requests
- Respond to requests:
  - Accept (full quantity)
  - Reject
  - Alter quantity (partial acceptance)
- Add optional notes/comments
- Quick action interface (minimal clicks)

---

## 7. Negotiation Module

### Features:
- Notify customer on vendor response
- Handle partial acceptance:
  - Customer accepts revised quantity
  - Customer cancels item
- Final order confirmation flow

---

## 8. Notification Module

### Features:
- In-app notifications
- SMS / WhatsApp notifications (optional integration)
- Event triggers:
  - New order request (to vendor)
  - Vendor response (to customer)
  - Reminder for pending responses

---

## 9. Admin Dashboard Module

### Features:
- View all orders
- Filter orders:
  - By customer
  - By vendor
  - By status
- View user list (customers/vendors)
- Activate / deactivate users
- Basic analytics:
  - Total orders
  - Top vendors
  - Top products

---

## 10. Order History Module

### Features:
- Customer:
  - View past orders
  - Reorder functionality
- Vendor:
  - View received requests history
- Filter by status/date

---

## 11. Settings & Configuration Module

### Features:
- Manage categories (Shirts, Jeans, etc.)
- Manage product attributes (fabric, pattern)
- Configure response time limits
- Platform-level controls (enable/disable features)

---

## 12. Media Management Module

### Features:
- Image upload handling
- Image compression
- Storage management
- Duplicate image handling (basic)
